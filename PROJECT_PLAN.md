# BreatheAI — План проекта

## Концепция
AI-коуч по дыхательным практикам. Помогает управлять состоянием через дыхание (box breathing, 4-7-8, когерентное дыхание и др.)

**Целевая аудитория:** Люди в стрессе, тревоге, с проблемами сна. Менеджеры, разработчики, спортсмены.

## Монетизация
- Telegram бот с Premium подпиской
- $5/мес → нужно 60 платящих пользователей → $300/мес
- Freemium: 3 сессии в день бесплатно, Premium — безлимит + расширенная аналитика

## Платформа
Telegram бот (Grammy + TypeScript) → возможно веб-версия позже

## Архитектура

```
Telegram User
     ↓
Grammy Bot (TypeScript)
     ↓
Claude Sonnet 4-6 (AI коуч)
     ↓
Redis (история диалогов)
     ↓
PostgreSQL + pgvector (база знаний техник)
```
